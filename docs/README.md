## üéØ Two-Tier System Overview

### Tier A: Self-Declaration (Under Threshold)
- **Process**: Simple declaration form
- **Verification**: Post-distribution spot checks only
- **Approval**: Same-day
- **Documentation**: Minimal - just basic ID

### Tier B: Detailed Review (Over Threshold)
- **Who**: Families exceeding basic thresholds
- **Extra allowed**: +$26,144 additional income
- **Process**: Full financial review
- **Timeline**: 5-10 business days
