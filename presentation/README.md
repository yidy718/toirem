## How to Use

### Quick Start

1. **Open the website:**
   - Double-click `index.html` to open in your browser
   - OR run `./launch.sh` in terminal (Mac/Linux)

2. **Navigate documents:**
   - Use the left sidebar to browse through all documentation tiers
   - Click any document to view it instantly
   - Currently selected document is highlighted in blue

3. **Use the calculator:**
   - Click "Universal Formula Calculator" in sidebar
   - Enter number of adults and children by age group
   - See instant calculation of thresholds

4. **Download as PDF:**
   - Click the "Download PDF" button in the top-right corner
   - The current view will be saved as a PDF file
