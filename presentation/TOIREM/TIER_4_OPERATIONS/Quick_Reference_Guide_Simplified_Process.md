## ‚úÖ SIMPLIFIED PROCESS (Under $5,000)

### Step 1: Check Basic Requirements
- [ ] Valid government ID?
- [ ] Lives in service area?
- [ ] Jewish Hasidic family?
- [ ] **SIGNED DECLARATION OF NEED? (This replaces income verification)**

### Step 2: Assess Observable Need
**Look for these indicators (no proof needed):**
- Large family (6+ children)
- Known to community/rabbi
- Living in modest housing
- Requesting necessities only
- Emergency timing

### Step 3: Quick Verification
- Check internal database for previous assistance
- Note family size and ages
- Confirm address is residential
- Look for any red flags

### Step 4: Approve/Process
- Fill out simplified worksheet
- Get manager signature
- Process payment (preferably direct to vendor)
- File documentation

**Timeline: 24-48 hours maximum**

---

## üìä FULL DOCUMENTATION (FULL DOCUIMENTS $5,000+)

### Required Documents Checklist:
- [ ] Government ID
- [ ] Proof of address
- [ ] Social Insurance Number
- [ ] Date of birth
- [ ] Completed full application
- [ ] Signed declaration
- [ ] Income verification (one of):
  - [ ] Notice of Assessment
  - [ ] T4 slips
  - [ ] Pay stubs (2 months)
  - [ ] Bank statements (2 months)
- [ ] Expense documentation
- [ ] Family size proof (CCB statement)

### Processing Steps:
1. Complete full income assessment
2. Calculate adjusted threshold (+40% for Hasidic families)
3. Verify all documentation
4. Get board approval (required over $5,000)
5. Process payment
6. Schedule follow-up

**Timeline: 5-10 business days**

## üî¥ RED FLAGS - INVESTIGATE
